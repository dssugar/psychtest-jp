# Phase 1 Report: SWLS (Satisfaction With Life Scale)

## Basic Information

- **Full Name**: Satisfaction With Life Scale
- **Abbreviation**: SWLS
- **Developer**: Ed Diener, Robert A. Emmons, Randy J. Larsen, Sharon Griffin
- **Year**: 1985
- **Journal**: Journal of Personality Assessment, 49(1), 71-75
- **DOI**: 10.1207/s15327752jpa4901_13

## What is SWLS?

SWLS（人生満足度尺度）は、**人生全体に対する認知的な満足度**を測定する心理尺度です。「現在の自分の人生をどの程度満足しているか」という主観的な評価を、わずか5項目で測定します。

**特徴**:
- **認知的判断**を測定（感情ではなく、思考による評価）
- **人生全体**に対する満足度（特定の領域ではなく）
- **短時間**で実施可能（約1分）

## Psychometric Properties

### 信頼性（Reliability）

**内部一貫性（Cronbach's α）**:
- **原著（Diener et al., 1985）**: α = 0.87
- **メタ分析（60研究）**: 平均 α = 0.78
- **各国研究**: α = 0.79-0.91
  - スウェーデン: α = 0.88
  - スペイン: α = 0.88
  - オランダ: α = 0.85
  - ギリシャ: α = 0.84
  - イラン: α = 0.887
  - コロンビア: α = 0.842
  - メキシコ: α = 0.79-0.80

**再テスト信頼性**:
- **2ヶ月**: r = 0.82
- **1ヶ月**: r = 0.84
- **4年**: r = 0.54
- **5年**: r = 0.51

**解釈**: 短期的には高い安定性、長期的にはライフイベントによる変化を反映して中程度の安定性。

### 妥当性（Validity）

**因子構造**:
- **単一次元構造**（1因子モデル）が確認されている
- 原著の因子分析で66%の分散を説明

**収束的妥当性**:
- 他の主観的幸福感尺度と中〜高程度の相関
- 不安尺度と負の相関: r = -0.410
- うつ病尺度と負の相関: r = -0.434

**予測的妥当性**:
- 性格特性（Big Five）と予測可能な相関
- 外向性、協調性、誠実性、情緒安定性と正の相関
- 神経症傾向と負の相関

### Items & Format

- **項目数**: 5項目
- **回答形式**: 7点リッカート尺度
  - 7 = 非常にそう思う (Strongly agree)
  - 6 = そう思う (Agree)
  - 5 = 少しそう思う (Slightly agree)
  - 4 = どちらでもない (Neither agree nor disagree)
  - 3 = あまりそう思わない (Slightly disagree)
  - 2 = そう思わない (Disagree)
  - 1 = 全くそう思わない (Strongly disagree)
- **得点範囲**: 5-35点
- **所要時間**: 約1分

### Sample Size & Citations

- **原著サンプル**: 初期テストで176名の大学生（48項目から5項目に絞り込み）
- **引用数**: 40,000+ (Google Scholar推定、世界で最も引用されている幸福感尺度の一つ)

## Psychological Layer

**Outcome（成果）** - 人生満足度・ウェルビーイング

**理由**:
- 人生満足度は、**特性（Trait）**と**状態（State）**の**結果**として生じる
- **特性の影響**: 性格特性（外向性、神経症傾向）が人生満足度を予測
- **状態の影響**: 現在のメンタルヘルス状態（不安、うつ）が満足度に影響
- **成果としての性質**: 人生全体に対する認知的評価であり、特性・状態が統合された最終的な「成果」

**他の尺度との関連**:
- **Big Five（Trait）** → 人格が満足度を予測
- **PHQ-9, GAD-7（State）** → 現在のメンタルヘルスが満足度に影響
- **Self-Esteem, Self-Concept（Outcome）** → 自己評価と人生満足度は相互関連

## Key Strengths

### 1. 簡潔性と効率性
わずか5項目で高い信頼性（α = 0.87）を達成している点は驚異的です。多くの心理尺度が10-20項目を必要とする中、SWLSは最小限の項目数で最大限の情報を得られます。

### 2. 文化横断的妥当性
世界中の多様な文化圏で翻訳・検証され、一貫して高い信頼性と妥当性が報告されています。西洋文化だけでなく、アジア、中東、南米でも使用可能です。

### 3. 理論的明確性
「人生全体に対する認知的満足度」という構成概念が明確で、感情的な幸福感（ポジティブ感情）や特定領域の満足度（仕事、家族）と区別されています。

### 4. 実用性
1分で実施可能なため、大規模調査、縦断研究、臨床現場での反復測定に最適です。

## Limitations & Considerations

### 1. 天井効果（Ceiling Effect）
一般集団では高得点に偏る傾向があり、特に幸福度の高い集団では変化を捉えにくい可能性があります。

### 2. 文化差
集団主義文化（日本など）では、個人主義文化（アメリカなど）よりもやや低いスコアになる傾向が報告されています。これは文化的価値観の違いを反映しています。

### 3. 一時的状態の影響
「認知的判断」を測定するとされていますが、現在の気分（State）の影響を完全には排除できません。特に、測定時点での一時的なネガティブイベントがスコアを下げる可能性があります。

### 4. 社会的望ましさバイアス
「人生に満足している」と答えることが社会的に望ましいと感じる回答者は、実際よりも高いスコアをつける可能性があります。

## Use Cases

### 1. 幸福研究（Well-Being Research）
ポジティブ心理学、幸福経済学、公衆衛生の研究で標準的に使用されます。

### 2. 介入効果測定
心理療法、コーチング、ウェルビーイング介入の効果を測定する際のアウトカム指標として最適です。

### 3. 疫学調査
大規模な国民健康調査、縦断研究で、人生満足度の推移や関連要因を調べます。

### 4. 組織調査
従業員満足度調査、ワークライフバランス調査で、全体的な生活満足度を把握します。

### 5. 臨床アセスメント
うつ病、不安障害の治療効果測定、QOL（Quality of Life）評価に使用されます。
